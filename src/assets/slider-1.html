問題與挑戰？

POCA 是我進修前端開發後完成的首個作品集，在專案中我全權負責前端開發工作。 與之前的形象網站相比，這次項目更加複雜和功能豐富，我運用了 React 框架實現各種功能，包括文章篩選、購物車、物件元件化以及資料處理，同時為了後續的維護，我會將重複的項目元件化，例如：文章與產品等。
在無後端支援的情況下，我會架構出 React copmponent 並以 object 的方式呼喚傳送資料，若後續文章或產品項目有需異動，可以節省維護時間，以及使用 localStorage 暫存資料，解決資料保存的問題。
使用第三方 API Mapbox 解決地圖顯示和交互問題，在網站中嵌入地圖並設計 POCA 獨特的咖啡 pin，增加網站的設計感與使用體驗。
同時我也發現使用者更喜歡使用標籤來搜尋內容，而不是依賴文章風格或發布時間。為此，我根據標籤數據新增了熱門標籤，提升了文章搜尋的便捷性。此外，我還注意到地址和菜單是常見的搜尋項目將其整合，解決咖啡廳資訊零散的問題，進一步提升使用者體驗。
在開發過程中，也不免遇到一些挑戰，查閱官方文件，正確使用 dependency，解決數據動態渲染錯誤的問題，並從中學習到了許多寶貴的經驗，這使我對前端設計開發有更深的理解與掌握，期待未來能參與不同的專案。
